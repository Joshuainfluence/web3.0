create two directories
-client
    where the react application will be
-smart_contract
    where solidity contract will go 

navigate to client directory
install react using the command -> npm init vite@latest
skip project name -> ./
select framework -> react
select variant-> javascript

install tailwindCSS using the commands -> 
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p


--------smart contract
navigate to the smart_contract directory
initialize an empty json package -> npm init -y

npm install react-icons ethers to install icons and the ethers is to enable interaction with the blockchain and smartcontract 


-----install all necesssary dependencies for smartcontract using the command
npm install --save-dev hardhat @nomiclabs/hardhat-waffle ethereum-waffle chai @nomiclabs/hardhat-ethers ethers
if facing any installation error, try runnning them one after the other

npm install --save-dev hardhat
npm install --save-dev @nomiclabs/hardhat-waffle ethereum-waffle
npm install --save-dev chai
npm install --save-dev @nomiclabs/hardhat-ethers ethers

after successful installation, run 
npx hardhat

if nothing works, run the code
npm install --save-dev --legacy-peer-deps hardhat @nomiclabs/hardhat-waffle ethereum-waffle chai @nomiclabs/hardhat-ethers ethers
npx hardhat
